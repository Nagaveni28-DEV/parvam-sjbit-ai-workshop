Prompt Engineering is a way of asking or interacting with a chatbot for getting the desired output.

Types of Prompts:
1) Zero Shot Prompt:
Ex: Write a python program to classify the even and odd numbers out of the given list of numbers.

Usecase :--- A customer supprt chatbot tot analyse the user emotions.

2) Few shot Prompt:
Ex: Write a python program to classify the even and odd numbers out of the given list of numbers.

For Example :[1,2,3,7,8,11,121,15,18]
Even=[2,8,12,18]
Odd=[1,3,7,11,15]

Usecase :--- Generating the product description by giving 3 to 4 sample descriptions.

3) Chain of Thoughts Prompts: (CoT)
Ex: write a python program to find the sum ogf given numbers using while loop with step by step explanation.

Usecase :--- to learn or explore new things with step by step explanation.

4} Self Consistency Prompt :(Extension of CoT)
Ex: Solve the problem with step by step reasoning and then verify : new problem

Usecase :--- Whenever the Accuracy needs to be high. ex: Medicxal opr financial advisory chatbot

5) Role/Persona Prompt:
Ex: You are a python expert, I need a python program to print the star pattern using loop..

Usecase :--- For proper expertise we can use this prompt.

6) System/Instruction Prompt: 
Ex: Always answer in one or 2 sentences for any question that I ask. 
What is the capital of India?

Usecase:--- to maintain the consistency in the answers for sharp answers

7)Negative Prompt: 
Ex: Explain about python without using the keyword: programming, scripting, AI, ML and similar to english.

Usdecase:--- for sensitive topics
